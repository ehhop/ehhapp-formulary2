    Invoices must have the following columns with these EXACT titles to be imported:

    Column title Description
    Item No The medication's unique identifier (used to match medication records together).
    Item Description The name of the medication
    Requisition Date The date that the order for medication was billed to EHHOP
    Issue Qty The amount of medication that was issued in this order.
    Extended Price The amount paid for the order by EHHOP
    "Comdty Name " (optional) The category name of the medication. Note that the extra space after the final character is important.
